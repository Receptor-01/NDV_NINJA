\# STEP 1 - NDV WORKFLOW - Creates a daily case folder.



\## Current Modules



\### 01\_create\_case\_folder.bat

\- Prompts for subject initials (F\_L)

\- Creates a folder on the user's Desktop

\- Format: A\_B-mmddyyyy

\- Terminal-only (no GUI dialogs)

\- Designed to be chained with future scripts



\## Usage



Double-click the script or run from Command Prompt: "scripts\\01\_create\_case\_folder.bat"



\## Design Goals



\- Modular

\- Chainable

\- Portable

\- GitHub-friendly

\- No admin privileges required
